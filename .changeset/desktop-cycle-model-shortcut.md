Add a "Cycle model" global shortcut that steps through available models, mirroring the existing tone cycler. Both the tone and model cycle shortcuts are now optional — they have no default and can be cleared from Settings. Shortcut recording now distinguishes the Ctrl key from the Windows/Super key (previously both were captured as Ctrl/⌘), so Super-based shortcuts can be bound on Windows and Linux.
